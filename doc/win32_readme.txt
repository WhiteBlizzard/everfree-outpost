Everfree Outpost (Windows Build)
================================

How to play:

 1. Run "server_gui.py" in this directory.  (Requires Python - see below)
 2. Click "Start Server".
 3. Open a web browser and navigate to http://localhost:8889/client.html

Other players can join by visiting http://(your IP address):8889/client.html.
You may need to forward ports 8888 and 8889 from your router.


# Python

The server GUI requires Python to run.  You can download Python from
http://python.org/.  Version 3.4+ is preferred, but 2.7 should also work.
